📐 Form Sequence Reference

Baseline (7 forms)
  1. Demographics / Lifestyle (incl. PGA at end)
  2. CAGE Conditional: drinks alcohol
  3. DLQI
  4. EuroQol EQ-5D + VAS
  5. HAQ Conditional: IA diagnosis
  6. HADS
  7. SAPASI (UX design TBD at sprint)
Follow-Up (up to 10 forms)
  1. PGA (first, standalone)
  2. Medical Problems Update
  3. DLQI
  4. EuroQol EQ-5D + VAS
  5. Smoking Update
  6. Drinking Update
  7. CAGE Conditional: drinks alcohol
  8. HADS
  9. HAQ Conditional: IA diagnosis
  10. SAPASI

⚙️ Key Design Decisions

EuroQol EQ-5D-3L 3 levels confirmed from PDF
PGA: 5-level categorical Clear → Almost clear → Mild → Moderate → Severe
HADS: all 14 items Q11–Q14 confirmed from PDF
SAPASI in v1 UX design decision at sprint
CAGE: conditional Only if patient drinks alcohol
HAQ: conditional Only if IA/PA diagnosis
SQLite for testing SQL Server for staging/prod
Tech stack .NET 10 / MAUI Blazor / Blazor Web
